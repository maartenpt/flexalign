# Command: `install`

Prints **pip install** hints for optional backend extras. Does not run pip itself.

```bash
flexalign install awesome
# → pip install "flexalign[awesome]"
```

## Arguments

- Optional **`backend`**: `awesome` | `hunalign` | `labse`

If omitted, prints help.
